## ✅ 第二步：下载开发者工具（10分钟）

```
https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
```

## ✅ 第三步：导入项目（5分钟）

### 操作步骤
1. 打开微信开发者工具
2. 点击"导入项目"或"+"号
3. 填写信息：
   - 项目名称：一日三餐
   - 目录：选择 `~/一日三餐/miniprogram`
   - AppID：填入你的 AppID
   - 开发模式：小程序
   - 后端服务：不使用云服务（暂时）

4. 点击"导入"

---

## ✅ 第四步：修改配置（10分钟）

### 4.1 修改 AppID
编辑 `miniprogram/project.config.json`:
```json
{
  "appid": "你的AppID"  // 替换这里
}
```

### 4.2 修改 API 地址
编辑 `miniprogram/utils/api.js`:
```javascript
// 修改这一行
apiBaseUrl: 'http://localhost:3000/api'  // 暂时先用本地测试
```

## ✅ 第五步：启动后端测试（10分钟）

### 5.1 启动 Node.js 后端
```bash
cd ~/一日三餐/server
npm install  # 如果还没安装依赖
npm run dev
```

### 5.2 测试 API
打开浏览器访问：

## ✅ 第九步：提交审核（10分钟）

### 操作步骤
1. 登录小程序后台：
   ```
   https://mp.weixin.qq.com
   ```

2. 开发管理 → 开发版本 → 提交审核

3. 填写审核信息：

   **功能页面1：首页**
   ```
   页面路径：pages/index/index
   功能描述：用户选择用餐类型（早餐/午餐/晚餐）和人数，点击生成菜单按钮
   ```

   **功能页面2：菜单页**
   ```
   页面路径：pages/menu/menu
   功能描述：展示AI生成的菜品列表，用户可以选择要制作的菜品
   ```

   **功能页面3：购物清单**
   ```
   页面路径：pages/shopping/shopping
   功能描述：根据选择的菜品自动生成食材清单，按分类展示
   ```

   **功能页面4：菜谱页**
   ```
   页面路径：pages/recipe/recipe
   功能描述：展示菜品的详细制作步骤、时间和烹饪小贴士
   ```

4. 上传截图（4张）

5. 填写补充说明：
   ```
   这是一款AI驱动的智能餐饮规划助手小程序。
   主要功能：
   1. 根据用户偏好智能生成菜单
   2. 自动生成购物清单
   3. 提供详细烹饪指导
   
   适用于需要规划日常饮食的用户，帮助用户快速决定吃什么、买什么菜、怎么做菜。
   ```

6. 点击"提交审核"
