# 📝 第1步完成指南：注册小程序账号

## 🔗 立即打开注册链接

```
https://mp.weixin.qq.com/wxopen/waregister?action=step1
```

---

## 📋 详细操作步骤（30分钟）

### 1. 填写账号信息
```
邮箱：___________________（建议使用常用邮箱）
密码：___________________（记住这个密码）
验证码：_________________（点击获取）
```

### 2. 邮箱激活
```
1. 登录你的邮箱
2. 找到来自"微信开放平台"的邮件
3. 点击激活链接
4. 完成激活
```

### 3. 信息登记
```
主体类型：选择【个人】（免费，无需营业执照）

个人信息：
- 姓名：___________________
- 身份证号：___________________
- 手机号：___________________
- 验证码：___________________

管理员扫码：用你的微信扫码确认
```

### 4. 获取 AppID（最重要！）
```
注册成功后：
1. 登录小程序后台：https://mp.weixin.qq.com
2. 左侧菜单：开发 → 开发管理
3. 点击：开发设置
4. 找到：AppID(小程序ID)

你的 AppID: ________________________

⚠️ 重要：把 AppID 记下来，后面要用！
```

---

## ⚠️ 常见问题

### Q: 邮箱已被注册？
**A**: 换一个邮箱，必须是未注册过微信开放平台的

### Q: 找不到 AppID？
**A**: 登录后台 → 开发 → 开发管理 → 开发设置

### Q: 个人主体有什么限制？
**A**: 
- ✅ 可以发布小程序
- ❌ 不能使用微信支付
- ❌ 某些类目不可选（如电商）

对于"一日三餐"项目，个人主体完全够用！
