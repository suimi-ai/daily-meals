# 小程序页面预览

## 页面结构

### 1. 首页 (pages/index)
**功能**：
- 选择用餐类型（早餐/午餐/晚餐）
- 设置用餐人数
- 生成今日菜单
- 快捷入口（热门推荐/我的收藏/历史菜单）

**UI元素**：
- 渐变色头部（#FF6B6B）
- 用餐类型选择器（3个按钮）
- 人数输入框
- 生成菜单按钮
- 3个快捷入口卡片

---

### 2. 菜单页 (pages/menu)
**功能**：
- 展示生成的菜品列表
- 选择要制作的菜品
- 查看营养信息
- 查看菜谱详情
- 生成购物清单

**UI元素**：
- 菜品卡片列表
  - 菜名、描述
  - 时间、难度
  - 选择状态（✓标记）
  - "查看菜谱"按钮
- 营养信息卡片（卡路里/蛋白质/碳水/脂肪）
- "生成购物清单"按钮

---

### 3. 购物清单页 (pages/shopping)
**功能**：
- 展示所需食材清单
- 按分类展示（肉类/蔬菜/调料等）
- 标记已购买状态
- 标记已有食材
- 复制购物清单

**UI元素**：
- 汇总信息卡片（总数/需购买/已有）
- 分类食材列表
  - 食材名称、分量
  - 状态标签（已有/需购买）
  - 购买建议
  - 点击标记已购买
- "复制购物清单"按钮

---

### 4. 菜谱页 (pages/recipe)
**功能**：
- 展示菜谱详细信息
- 步骤导航（上一步/下一步）
- 查看烹饪小贴士
- 查看营养信息

**UI元素**：
- 头部卡片（菜名、描述、时间、难度）
- 食材清单
- 步骤指示器（1-2-3-4...）
- 当前步骤详情
  - 步骤标题、时间
  - 详细描述
  - 小贴士（💡）
- 上一步/下一步按钮
- 营养信息
- 烹饪小贴士列表

---

## 主题色

- **主色**：#FF6B6B（珊瑚红）
- **辅助色**：#52c41a（绿色-已有）
- **背景色**：#f5f5f5
- **文字色**：#333（主要）、#666（次要）、#999（辅助）

---

## 交互说明

1. **首页**
   - 点击用餐类型按钮切换选中状态
   - 输入人数（1-10）
   - 点击"生成今日菜单"调用AI生成

2. **菜单页**
   - 点击菜品卡片选择/取消选择
   - 点击"查看菜谱"跳转到菜谱页
   - 选择菜品后点击"生成购物清单"

3. **购物清单页**
   - 点击食材卡片标记已购买（透明度降低）
   - 点击"复制购物清单"复制到剪贴板

4. **菜谱页**
   - 点击步骤指示器跳转到指定步骤
   - 点击"上一步/下一步"导航
   - 查看每个步骤的小贴士

## 如何预览

### 方法1：微信开发者工具
1. 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开工具，选择"导入项目"
3. 项目路径：`~/一日三餐/miniprogram`
4. AppID：选择"测试号"或使用自己的AppID
5. 点击"编译"即可预览

### 方法2：截图模拟
（需要在微信开发者工具中运行后截图）

## 注意事项

1. **API配置**：需要先启动后端服务
   ```bash
   cd ~/一日三餐/server
   npm install
   npm run dev
   ```

2. **修改API地址**：编辑 `miniprogram/utils/api.js`
   ```javascript
   apiBaseUrl: 'http://localhost:3000/api'
   // 或改为实际服务器地址
   ```

3. **微信开发者工具设置**：
   - 详情 → 本地设置 → 勾选"不校验合法域名"
   - 这样才能访问localhost
