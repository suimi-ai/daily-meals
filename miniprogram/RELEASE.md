# 微信小程序发布指南

## 第一步：注册小程序账号

### 1. 访问注册页面
https://mp.weixin.qq.com/wxopen/waregister?action=step1

### 2. 填写信息
- 邮箱（未注册过微信开放平台的邮箱）
- 密码
- 验证码

### 3. 邮箱激活
- 登录邮箱，点击激活链接

### 4. 信息登记
- 主体类型：个人/企业
- 个人：身份证信息
- 企业：营业执照等

### 5. 获取 AppID
注册成功后，在「开发」→「开发管理」→「开发设置」中可以看到 AppID

---

## 第二步：配置项目

### 1. 修改 app.json
```json
{
  "pages": [...],
  "window": {
    "navigationBarTitleText": "一日三餐",
    ...
  }
}
```

### 2. 获取 AppID 后修改
在微信开发者工具中：
- 项目详情 → AppID → 输入你的 AppID

---

## 第三步：开发调试

### 1. 下载开发者工具
https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 导入项目
- 选择项目目录：`~/一日三餐/miniprogram`
- 输入 AppID
- 点击「导入」

### 3. 调试测试
- 在模拟器中测试所有功能
- 真机调试（扫码在手机上测试）
- 修复所有 bug

---

## 第四步：上传代码

### 1. 上传
在微信开发者工具中：
- 点击右上角「上传」按钮
- 填写版本号：1.0.0
- 填写项目备注：初始版本

### 2. 登录后台
https://mp.weixin.qq.com

### 3. 查看版本
开发管理 → 开发版本 → 可以看到上传的版本

---

## 第五步：提交审核

### 1. 设置基本信息
设置 → 基本设置
- 小程序名称：一日三餐
- 小程序简介：AI驱动的智能餐饮规划助手
- 服务类目：生活服务 > 美食 > 菜谱
- 头像：上传图标

### 2. 提交审核
开发管理 → 开发版本 → 提交审核

填写审核信息：
- 功能页面：首页、菜单页、购物页、菜谱页
- 测试账号：（如果需要）
- 补充说明：AI生成菜单，智能购物清单，烹饪指导

### 3. 等待审核
- 通常 1-7 个工作日
- 可在后台查看审核进度

---

## 第六步：发布上线

### 1. 审核通过后
开发管理 → 审核版本 → 发布

### 2. 全量发布
- 点击「全量发布」
- 所有用户都能访问

---

## 注意事项

### ⚠️ 域名配置（重要！）

1. 登录小程序后台
2. 开发 → 开发管理 → 开发设置 → 服务器域名
3. 添加 request 合法域名：
   ```
   https://你的服务器域名
   ```

### ⚠️ 隐私协议

设置 → 基本设置 → 服务内容声明
- 填写用户隐私保护指引

### ⚠️ 敏感内容

避免包含：
- 政治敏感内容
- 侵权内容
- 虚假宣传

## 后端服务器部署

小程序需要 HTTPS 域名，你需要：

### 选项 1：云服务器
- 阿里云 / 腾讯云 / 华为云
- 配置域名和 SSL 证书
- 部署 Node.js 服务

### 选项 2：Serverless
- 微信云开发
- 腾讯云 SCF
- 阿里云函数计算

---

## 需要准备的材料

1. **营业执照**（如果是企业）
2. **身份证**（个人开发者）
3. **小程序图标** 1024x1024
4. **服务器域名** + SSL 证书
5. **AppID**（注册后获取）

- 注册账号：1-2 天（邮箱激活 + 信息登记）
- 开发调试：1-3 天
- 提交审核：1-7 天
- **总计：3-12 天**
