# 📝 第2步完成指南：下载工具和导入项目

## 🔗 下载开发者工具

### 下载地址
```
https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
```

### 选择版本
- **macOS**: 下载 `wechat_devtools_x.x.x_x64.dmg`
- **Windows**: 下载 `wechat_devtools_x.x.x_x64.exe`
- **Linux**: 下载对应版本

---

## 📥 安装步骤（10分钟）

### macOS
```
1. 打开下载的 .dmg 文件
2. 拖动"微信开发者工具"到"应用程序"
3. 打开"应用程序"文件夹
4. 双击"微信开发者工具"
5. 首次打开会提示"无法验证开发者"，点击"打开"
```

### Windows
```
1. 双击下载的 .exe 文件
2. 点击"是"允许安装
3. 选择安装位置（默认即可）
4. 点击"安装"
5. 完成后点击"启动"
```

## 📂 导入项目（5分钟）

### 启动开发者工具
```
1. 打开"微信开发者工具"
2. 使用微信扫码登录
3. 点击左侧的"+"号或"导入项目"
```

### 填写项目信息
```

目录：/Users/openclaw/一日三餐/miniprogram
      或者点击"选择"按钮，找到这个文件夹

```

### 导入成功
```
点击"导入"按钮
等待项目加载完成
看到模拟器显示首页 = 成功！
```

---

## 🎨 项目配置（5分钟）

### 修改 AppID（如果之前没改）
编辑文件：`miniprogram/project.config.json`

找到这一行：
```json
"appid": "请替换为你的AppID"
```

改为：
```json
"appid": "wx你的AppID"
```

### 保存并重新编译
```
1. 按 Cmd+S (Mac) 或 Ctrl+S (Windows) 保存
2. 点击开发者工具顶部的"编译"按钮
3. 查看模拟器是否正常显示
```

## ⚠️ 常见问题

### Q: 导入时提示"AppID 不存在"？
**A**: 检查 AppID 是否正确，必须是 wx 开头的字符串

### Q: 模拟器显示空白？
**A**: 
1. 检查 project.config.json 中的 AppID
2. 点击"编译"重新加载
3. 查看控制台是否有错误

### Q: 点击按钮没反应？
**A**: 
1. 检查后端是否启动
2. 查看控制台网络请求
3. 确认 API 地址正确

### Q: 真机调试无法连接？
**A**: 
1. 确保手机和电脑在同一 WiFi
2. 检查防火墙设置
3. 尝试关闭防火墙
