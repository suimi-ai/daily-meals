# 一日三餐 🍳

AI驱动的智能餐饮规划助手

## 项目简介

通过AI为用户提供个性化的餐饮规划服务，包括菜单推荐、购物清单生成和烹饪指导。

## 核心功能

### 1. 智能菜单生成 📋
- 根据用户输入（偏好、食材、人数等）生成早餐/午餐/晚餐菜单
- 提供多种菜品选择供用户挑选
- 支持个性化推荐

### 2. 智能购物清单 🛒
- 根据选择的菜品自动生成购物清单
- 包含食材名称、建议分量
- 标注现有食材（可选）
- 提供购买建议

### 3. 烹饪指导 👨‍🍳
- 根据用户选择的菜品提供详细制作流程
- 可选择性查看特定菜品的做法
- 包含步骤说明、时间预估等
